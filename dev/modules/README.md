# Modules Folder

This folder contains the written course material for individual modules.

Each module is stored as its own Markdown file and is numbered in sequence so the course structure is easy to follow.

## Current Contents

- [01_introduction_to_process_control.md](./01_introduction_to_process_control.md): the first module, based on the source chapters on process control fundamentals, feedback, and control objectives.

Additional module files can be added here later using the same naming pattern:

- `02_next_module_name.md`
- `03_next_module_name.md`
